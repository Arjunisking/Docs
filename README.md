# AI Website Planning Docs

A clean documentation system for planning and building better websites with AI tools.

This repository contains four core documents that help AI builders generate more accurate, structured, and production-ready websites:

- PRD
- Design Document
- Tech Stack Document
- How to Use Guide

## Why this repository exists

Most AI-generated websites turn out generic because the instructions are vague.

This repository fixes that by giving AI the three things it actually needs:

- Clear product requirements
- Clear visual direction
- Clear technical constraints

When these documents are used together, AI tools can produce better website structure, design consistency, and implementation quality.

Documents included
1. PRD

The Product Requirement Document defines what the website should do.

It includes:

Project overview
Goals
Target audience
Pages
Features
User journey
Success metrics

Use this when you want the AI to understand the product and business side of the website.

2. Design Document

The Design Document defines how the website should look and feel.

It includes:

Visual style
Mood
Color palette
Typography
Layout system
Component design
Page-specific design guidance

Use this when you want the AI to follow a clear visual direction instead of generating generic layouts.

3. Tech Stack Document

The Tech Stack Document defines what technologies should be used.

It includes:

Frontend framework
Meta-framework
Styling solution
Backend options
Libraries and packages
Deployment recommendations
Performance and SEO tools

Use this when you want the AI to build with the right stack and avoid random technology choices.

4. How to Use Guide

This file explains how to use all of these documents together with AI tools.

It includes:

How to prepare the files
Which AI tools can use them
How to prompt effectively
How to build step by step
How to review the final output
Recommended workflow
Customize the PRD for your project
Customize the Design Document
Customize the Tech Stack Document
Review the How to Use guide
Paste or attach the documents into your AI tool
Ask the AI to start with homepage and navigation
Build page by page
Compare the final result against the documents
Example prompt

Use this inside ChatGPT, Claude, Bolt, Cursor, v0, or similar tools:

I want to build a website based on these documents:

[Attach or paste PRD]
[Attach or paste Design Document]
[Attach or paste Tech Stack Document]

Please analyze these documents and create a complete website that:
1. Implements all features from the PRD
2. Follows the design specifications closely
3. Uses the specified tech stack
4. Is production-ready and well-structured

Start with the homepage and core navigation.
